# bnb-lpl

**bnb-lpl** helps you find the highest yield for a specific asset, on [Binance Launchpool](https://launchpad.binance.com/en/tab3).

## Usage

```shell script
// Install the dependencies
$ pip install -r requirements.txt

// Run the script
$ py main.py
```
